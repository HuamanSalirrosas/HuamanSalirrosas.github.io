# Página de prueba

Este sitio tiene dos páginas muy sencillas:

- `index.html` muestra una imagen. La etiqueta `<a>` hace que se pueda pulsar.
- `test.html` muestra el mensaje «esto es una prueba».
- `styles.css` adapta la imagen al alto visible en pantallas de PC y móvil,
  mostrándola completa y centrada en dos tercios del espacio disponible, sin
  recortarla ni añadir desplazamiento innecesario.

Para verlo, abre `index.html` en un navegador y pulsa la imagen.
