# Página de prueba

Este sitio tiene dos páginas muy sencillas:

- `index.html` precarga y muestra una imagen SVG no interactiva y, a su derecha,
  los enlaces «Viajes» y «Libros».
- `test.html` muestra el mensaje «esto es una prueba».
- `styles.css` muestra la imagen completa, sin recortarla, en dos tercios del
  ancho disponible. Si su altura supera la ventana, la página conserva la barra
  de desplazamiento para que se pueda recorrer el retrato entero.

Para verlo, abre `index.html` en un navegador y usa los enlaces situados junto a
la imagen.
